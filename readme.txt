Для установки проекта, установить зависимости для бэкенда и фронтенда и общие, выполнить последовательно:
  1. cd backend && npm i
  2. cd ../frontend && npm i
  3. cd ../ && npm i

Для запуска, из топ директории запустить:
   npm run start
   (запустятся оба сразу, бэк и фронт)

Посты сохранены в файле posts.json на бэке, меняются только в переменной(при перезагрузке бэка загрузятся те же). 

Работоспособность проверена на Chrome, Macbook,
Для клавиши Del так же используются аналоги 'Delete' и 'Backspace'.
Для фокусировки поста добавлен border, для наглядности.
